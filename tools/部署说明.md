# 教学工具页面部署说明

## 概述

本页面用于展示小学数学教学工具，支持自动读取 `tools/` 文件夹中的 HTML 工具文件，并以卡片形式展示。

## 添加新工具步骤

### 第一步：准备工具文件

将你的 HTML 教学工具文件放入 `tools/` 文件夹中。文件名可以使用英文或中文，建议使用有意义的文件名。

示例：
```
tools/
├── calculator.html
├── score management.html
└── 你的新工具.html
```

### 第二步：配置工具信息

编辑 `_data/tools.yml` 文件，添加新工具的配置信息：

```yaml
- filename: 你的工具文件名.html
  emoji: 🎮
  title: 工具中文名称
  description: 工具的使用场景说明，1-2行即可
```

配置项说明：
- `filename`: 工具文件名，必须与 `tools/` 文件夹中的文件名完全一致
- `emoji`: 工具图标，选择一个合适的 emoji
- `title`: 工具中文标题，让小学生能看懂
- `description`: 使用场景说明，描述工具的用途和适合的学习场景

### 第三步：预览效果

启动本地服务器预览效果：

```bash
bundle exec jekyll serve
```

然后访问 `http://localhost:4000/tools.html` 查看新工具是否正确显示。

### 第四步：部署到 GitHub

提交代码到 GitHub，GitHub Pages 会自动构建部署。

## 配置示例

```yaml
- filename: math-game.html
  emoji: 🎯
  title: 数学闯关游戏
  description: 通过有趣的闯关游戏，帮助小朋友练习加减法运算。
```

## 注意事项

1. HTML 文件必须放在 `tools/` 文件夹内
2. 文件名必须与 `_data/tools.yml` 中的 `filename` 完全匹配
3. 工具卡片会自动按两列布局显示，适配手机和电脑端
4. 建议每个工具的描述控制在 2 行以内，保持页面整洁
5. 选择适合小学生理解的 emoji 图标和中文标题
